# Loplex Wayland-tap

## How do I install these formulae?

`brew install loplex/wayland-tap/<formula>`

Or `brew tap loplex/wayland-tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "loplex/wayland-tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
